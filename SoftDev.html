What is a Software Developer?
A software developer is someone who creates, designs and maintains software applications. They use programming languages such as Python, Java, JavaScript and many more to build applications for computers, mobile devices and the web. These applications could be things such as games like: Elden Ring, minecraft, GTA and so on. Or it could be tools used by different careers like in hospitals such as Laerdal which is used to simulate certain scenarios on how to treat patients.

Data Analyst

Data analysts collect, process and analyse data to help organisations make informed decisions. They use various tools and techniques to interpet data, identify patterns, trends and insight. With these findings, they are presented in a clear and understandable way. Data analysts work with many different types of data such as: sales data, customer data, operational data and so on. Data Analysts use quite a few tools and programming languages like: Excel, SQL, Python and much more.

Cybersecurity

Cybersecurity is the practice of protecting computer systems, networks, and data from unauthorized access, attacks, or damage. It involves using tools, techniques, and best practices to safeguard sensitive information from hackers, malware, and other cyber threats. Key areas include network security, data encryption, ethical hacking, and security awareness. For beginners, learning cybersecurity means understanding how to secure devices, recognize online threats, and follow safe practices to prevent cyberattacks.

Cloud Computing & DevOps

Cloud computing is the delivery of computing services. This includes: servers, storage, databases, networking, software and analytics over the internet. The term "cloud" simply refers to a network of remote servers accessed over the internet that store, manage and process data instead of using a local computer or on-premises server. Simply put, it is just someone else's computer that you access online which you can use to perform certain tasks like storing files, running apps or performing tasks. Examples of these would be: Google drive, dropbox and Netflix are clouds and all rely on cloud infrastructure. DevOp
